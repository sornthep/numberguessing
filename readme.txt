This is a javascript based web app. An exercise from a javascript course. Consisting of 3 files: 
- index.html
- style.css 
- script.js 

The game: user try to guess a computer generated secret number (between 1 and maxi). 
The user's score is decreased every wrong guess. When a correct guess is made, the score
is finalized and the highest score is updated (kept until the app is closed.)
